1️⃣ What is the difference between var, let, and const?
Ans:
All 3 of them are used to declare variables, but the catch is:
Var: It is old version and it's value can be redeclared.
let: It can be reassigned by can not be redeclared. 
const: It can not be reassigned, nor redeclared. 

2️⃣ What is the spread operator (...)?
Ans: 
It is used to expand/ unpack values from arrays, objects or elements. 

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:
map(), is used to create new array by changing the items.
filter(), is used for selecting some items in an array based on any condition.
forEach(), it acts to do something with each item of an array but does not want a new array returned.
